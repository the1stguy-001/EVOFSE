# 🔬 Project: EVOFSE

> A step-by-step coding journey exploring the wonders of science.

Welcome! This is a web project dedicated to making scientific concepts accessible and exciting. It marks my first hands-on experience learning to code from the ground up, built in collaboration with AI.

---

## 🎯 Project Overview
Instead of relying purely on automated drag-and-drop templates, I wanted to dive into the actual code. I constructed the core architecture, designed the foundational layouts, and meticulously detailed the content to bring this science hub to life. 

*   **The Goal:** Create a clean, educational space for science enthusiasts.
*   **The Approach:** Code step-by-step, understanding the mechanics behind the screen.
*   **AI Collaboration:** Used as a coding peer to debug, brainstorm, and optimize performance.

---

## 🛠️ Built With

*   **HTML5** – Structuring the core elements and semantic text.
*   **CSS3** – Styling layout design, custom colors, and responsiveness.
*   **AI Assistant** – Guiding logic, troubleshooting bugs, and explaining complex code.

---

## 💡 What I Learned
Building this site taught me foundational web mechanics that are essential for any developer:
1. **Semantic Layouts:** How to structure pages logically so they are accessible.
2. **Detail Orientation:** Injecting targeted content while managing clean file structures.
3. **Iterative Problem Solving:** Breaking down complex design requirements into simple code instructions.
